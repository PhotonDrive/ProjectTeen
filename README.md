  b. Arduino - https://www.arduino.cc/
  c. ESP32-32 - https://www.espressif.com/en/products/hardware/development-boards
  d. Many more.....

3. Software
  a. Node JS - https://nodejs.org/en/about/
  b. Rust - https://www.rust-lang.org/
  c. Go  - https://golang.org/
